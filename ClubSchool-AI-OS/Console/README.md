# ClubSchool AI OS — 웹 콘솔

브라우저에서 ClubSchool AI OS의 디지털 조직과 직접 일하는 단일 페이지 콘솔입니다.
서브에이전트와 대화하고, 커맨드/워크플로우를 실행 프롬프트로 만들고, Gold Wiki와 산출물을
바로 보고, 10단계 품질 검증까지 한 화면에서 수행합니다.

## 주요 기능

| 기능 | 설명 |
|------|------|
| 🤖 에이전트 | 활성 서브에이전트 카드 → 상세 보기 → **바로 대화** |
| ⌘ 커맨드 | 슬래시 커맨드 선택 → 인자 입력 → 실행 프롬프트 생성/실행 |
| 🔀 워크플로우 | 핵심 워크플로우 런북 렌더링 |
| 📘 Gold Wiki | 토픽 폴더 + 번호형 문서 트리 탐색 · 마크다운 렌더링 |
| 📄 템플릿 / 📦 산출물 예시 | 복사용 템플릿과 모범 산출물 열람 |
| ✅ 품질 검증 | 10단계 품질 검증 체계 체크리스트 · 점수 · 보고서(.md) 내보내기 |
| ⚙️ 설정 | (선택) Anthropic API 키로 브라우저에서 직접 대화 |

## 실행 방법

콘솔은 정적 파일이며, 상위 폴더의 문서를 읽기 위해 **ClubSchool-AI-OS 루트에서 서빙**해야 합니다.

```bash
# 1) (문서가 바뀌었으면) manifest 갱신
cd ClubSchool-AI-OS
python3 Console/build-manifest.py

# 2) 정적 서버 실행 (루트에서)
python3 -m http.server 8000

# 3) 브라우저에서 열기
#    http://localhost:8000/Console/
```

> Node 환경이면 `npx serve .` 등 어떤 정적 서버도 가능합니다. 반드시 `ClubSchool-AI-OS/`를
> 루트로 서빙하고 `/Console/` 경로로 접속하세요(문서가 `../GoldWiki/...`로 로드됩니다).

## 두 가지 사용 모드

1. **프롬프트 모드(기본, 키 불필요)** — 입력한 작업을 에이전트/커맨드 정의 + Gold Wiki 거버넌스와
   결합한 **실행 프롬프트**로 만들어 클립보드에 복사합니다. 이를 Claude Code나 claude.ai에 붙여넣어 실행.
2. **직접 대화 모드(API 키 입력)** — 설정에 Anthropic API 키를 넣으면, 선택한 에이전트의 정의를
   시스템 프롬프트로 사용해 브라우저에서 바로 응답을 받습니다. 응답은 작업 패널에 렌더링됩니다.

## 보안 메모

- API 키는 브라우저 `localStorage`에만 저장되며 서버로 전송되지 않습니다.
- 직접 대화는 `anthropic-dangerous-direct-browser-access` 헤더를 사용합니다. 개인 PC/사내망에서만
  사용하고, 공용 환경에서는 프롬프트 모드를 권장합니다.

## 파일 구성

```
Console/
├── index.html          UI 셸
├── styles.css          스타일(디자인 토큰 기반)
├── app.js              로직(렌더링·대화·검증)
├── build-manifest.py   manifest.json 생성기(저장소 스캔)
├── manifest.json       에이전트/커맨드/문서 인덱스(생성물)
└── README.md
```

마크다운 렌더링은 `marked` + `DOMPurify`(CDN)를 사용합니다. 오프라인 환경에서는 자동으로 최소
렌더러로 폴백합니다. 향후 v2.0에서 오케스트레이터 백엔드와 연동해 실시간 멀티에이전트 실행을
지원할 예정입니다(`../Docs/architecture-v2/` 참조).
