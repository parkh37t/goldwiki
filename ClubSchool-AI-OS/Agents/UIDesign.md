# UIDesign — UI 디자인팀 역할 카탈로그

> 시각·인터랙션·모션·반응형 UI 디자인을 담당하는 사람용 역할 카탈로그입니다. 실행 정본은
> `../.claude/agents/ui-design-lead.md`, 거버넌스·표준 정본은
> [GoldWiki/UI](../GoldWiki/UI/README.md) 및 [GoldWiki/DesignSystem](../GoldWiki/DesignSystem/README.md)입니다.

## 팀 개요

UI 디자인팀은 프로덕트·UX팀의 구조와 플로우를 실제로 보고 만지는 화면으로 구현합니다.
시각 위계·인터랙션·모션·반응형을 통해 사용성과 심미성을 동시에 달성하며, 모든 디자인은
[DesignSystem](../GoldWiki/DesignSystem/README.md)의 토큰·컴포넌트를 우선 사용하고 일탈 시
근거를 남깁니다. 접근성(WCAG)과 구현 가능성을 항상 함께 고려합니다.

- 핵심 산출물: 고해상도 UI 시안, 인터랙션 명세, 모션 가이드, 반응형 규칙, 핸드오프 스펙
- 주요 협업 팀: 프로덕트·UX팀, 디자인시스템팀, 브랜드경험팀, 퍼블리싱·프론트팀
- 핵심 문서: [UI](../GoldWiki/UI/README.md), [DesignSystem](../GoldWiki/DesignSystem/README.md), [UX](../GoldWiki/UX/UXStrategyFramework.md)

## UI 디자인 리드 (UI Design Lead)

- 미션: UI 디자인의 방향·품질·일관성을 책임지고 시각 언어가 제품 전략과 브랜드를 충실히 구현하게 한다.
- 주요 책임: 디자인 컨셉/비주얼 디렉션 / 시안 품질 리뷰·승인 / 디자인 시스템 준수 강제 / 핸드오프 표준 관리 / 디자이너 작업 조율.
- 입력: 와이어프레임, IA, 브랜드 가이드, 디자인 시스템.
- 출력: 비주얼 디렉션, 승인된 UI 시안, 핸드오프 스펙, 디자인 QA 결과.
- 협업 대상: 프로덕트 오너, DS 아키텍트, BX 디렉터, 퍼블리싱 리드.
- 품질 기준: 시스템 정합성, 접근성 충족, 화면-요구 추적, 구현 가능성 검증.

## 비주얼 디자이너 (Visual / UI Designer)

- 미션: 레이아웃·타이포·컬러·아이콘으로 명료하고 매력적인 화면 시각 위계를 설계한다.
- 주요 책임: 화면 시안 제작 / 시각 위계·그리드 설계 / 컬러/타이포 적용 / 상태(state)별 디자인 / 디자인 시스템 컴포넌트 활용.
- 입력: 와이어프레임, 디자인 토큰, 콘텐츠, 브랜드 가이드.
- 출력: 고해상도 UI 시안, 상태별 화면, 에셋, 스타일 적용 노트.
- 협업 대상: UI 디자인 리드, 컴포넌트 디자이너(DS), 인터랙션 디자이너, 퍼블리셔.
- 품질 기준: 그리드/간격 일관, 대비(WCAG AA) 충족, 토큰 사용, 픽셀 정합.

## 인터랙션 디자이너 (Interaction Designer, IxD)

- 미션: 사용자 입력에 대한 시스템 반응을 설계해 직관적이고 예측 가능한 상호작용을 만든다.
- 주요 책임: 인터랙션 패턴 정의 / 상태/피드백(hover·focus·error 등) 설계 / 마이크로인터랙션 / 입력/제스처 처리 / 접근성 상호작용.
- 입력: 유저플로우, 와이어프레임, UI 시안, 컴포넌트 명세.
- 출력: 인터랙션 명세, 상태 다이어그램, 마이크로인터랙션 가이드, 프로토타입.
- 협업 대상: 유저플로우 디자이너, 비주얼 디자이너, 모션 디자이너, 프론트 리드.
- 품질 기준: 상태 누락 0, 피드백 즉시성, 키보드/스크린리더 대응, 패턴 일관.

## 모션 디자이너 (Motion Designer)

- 미션: 전환·애니메이션을 통해 흐름의 맥락과 브랜드 감성을 전달하고 사용성을 강화한다.
- 주요 책임: 모션 원칙·타이밍 정의 / 전환/로딩/피드백 애니메이션 / 모션 토큰화 / 성능/접근성 고려 / 모션 가이드 작성.
- 입력: 인터랙션 명세, 브랜드 모션 가이드, UI 시안, 성능 제약.
- 출력: 모션 가이드, 애니메이션 스펙(타이밍·이징), 프로토타입, 모션 토큰.
- 협업 대상: 인터랙션 디자이너, 토큰 엔지니어(DS), 프론트 리드, BX 디렉터.
- 품질 기준: 타이밍/이징 일관, prefers-reduced-motion 대응, 성능 영향 검증.

## 반응형 디자이너 (Responsive / Adaptive Designer)

- 미션: 다양한 디바이스·뷰포트에서 일관되고 최적화된 레이아웃을 설계해 어디서나 좋은 경험을 보장한다.
- 주요 책임: 브레이크포인트 전략 / 반응형 그리드/레이아웃 / 적응형 컴포넌트 동작 / 터치/입력 적응 / 디바이스별 QA.
- 입력: 화면 시안, 그리드 토큰, 콘텐츠 우선순위, 대상 디바이스.
- 출력: 반응형 규칙서, 브레이크포인트별 시안, 적응 동작 명세, 디바이스 QA 결과.
- 협업 대상: 비주얼 디자이너, IA, 퍼블리싱 리드, 프론트 리드.
- 품질 기준: 모든 브레이크포인트 정의, 콘텐츠 가독성 유지, 터치 타깃 기준 충족.

## UI 프로토타이퍼 (UI Prototyper)

- 미션: 인터랙티브 프로토타입으로 디자인 의도를 검증·전달해 이해관계자와 개발팀의 오해를 없앤다.
- 주요 책임: 클릭형/고충실 프로토타입 제작 / 사용성 테스트용 시나리오 구현 / 디자인 의도 시연 / 피드백 반영 / 핸드오프 보조.
- 입력: UI 시안, 인터랙션/모션 명세, 테스트 시나리오.
- 출력: 인터랙티브 프로토타입, 시연 자료, 테스트 반영 변경분.
- 협업 대상: UI 디자인 리드, 인터랙션 디자이너, UX 리서처, 프론트 리드.
- 품질 기준: 핵심 플로우 재현, 인터랙션 충실, 최신 시안 동기화.
