# BXBrand — 브랜드경험(BX)팀 역할 카탈로그

> 브랜드 전략·아이덴티티·보이스·가이드라인으로 일관된 브랜드 경험을 설계하는 사람용 역할
> 카탈로그입니다. 실행 정본은 `../.claude/agents/bx-design-lead.md`, 거버넌스·표준 정본은
> [GoldWiki/Brand](../GoldWiki/Brand/README.md)입니다.

## 팀 개요

브랜드경험팀은 제품·서비스의 모든 접점에서 일관되고 차별화된 브랜드 인식을 만듭니다.
브랜드 전략과 비주얼/언어 아이덴티티를 정의하고, 이를 디자인시스템·UI·마케팅 산출물에
적용해 신뢰와 호감을 축적합니다. 모든 표현은 [Brand](../GoldWiki/Brand/README.md) 가이드라인을
먼저 참조하고, 일탈은 근거와 함께 기록합니다.

- 핵심 산출물: 브랜드 전략, 비주얼 아이덴티티(VI), 보이스&톤, 브랜드 가이드라인, 적용 에셋
- 주요 협업 팀: 경영/리더십팀, 디자인시스템팀, UI 디자인팀, 영업·제안팀
- 핵심 문서: [Brand](../GoldWiki/Brand/README.md), [DesignSystem](../GoldWiki/DesignSystem/README.md), [UI](../GoldWiki/UI/README.md)

---

## BX 디렉터 (Brand Experience Director)

- 미션: 브랜드 경험 전체의 방향·일관성·품질을 책임지고 모든 접점에서 브랜드 약속이 지켜지게 한다.
- 주요 책임: 브랜드 비전/경험 원칙 정의 / 접점 일관성 감독 / 브랜드 산출물 승인 / 브랜드-비즈니스 정렬 / 브랜드 리스크 관리.
- 입력: 회사 전략, 브랜드 전략, 시장/경쟁 인사이트, 디자인 시스템.
- 출력: 브랜드 경험 원칙, 디렉션, 승인된 브랜드 산출물, 일관성 감사 결과.
- 협업 대상: 대표/총괄PD, 브랜드 전략가, VI 디자이너, DS 아키텍트.
- 품질 기준: 전략 정합성, 접점 일관, 차별화 명확, 가이드라인 준수.

## 브랜드 전략가 (Brand Strategist)

- 미션: 시장·고객·경쟁 분석을 토대로 브랜드 포지셔닝과 핵심 메시지 체계를 설계한다.
- 주요 책임: 브랜드 포지셔닝/가치 정의 / 타깃·페르소나 / 메시지 아키텍처 / 브랜드 차별화 전략 / 브랜드 성과 지표.
- 입력: 시장/경쟁 리서치, 비즈니스 목표, 사용자 인사이트, 가치 모델.
- 출력: 브랜드 전략서, 포지셔닝 맵, 메시지 하우스, 브랜드 KPI.
- 협업 대상: BX 디렉터, 전략기획 책임자, 경쟁 분석가, 카피/보이스 담당.
- 품질 기준: 근거 기반, 차별화 명확, 메시지 일관, KPI 측정 가능.

## 비주얼 아이덴티티 디자이너 (Visual Identity Designer)

- 미션: 로고·컬러·타이포·그래픽 요소로 즉시 알아보고 기억되는 시각 아이덴티티를 구축한다.
- 주요 책임: 로고/심볼 디자인 / 브랜드 컬러/타이포 시스템 / 그래픽 모티프/일러스트 / 적용 예시 / 아이덴티티 일관성.
- 입력: 브랜드 전략, 포지셔닝, 디자인 토큰 요건, 적용 매체.
- 출력: 로고 시스템, 컬러/타이포 규정, 그래픽 요소, VI 적용 에셋.
- 협업 대상: BX 디렉터, 브랜드 전략가, 토큰 엔지니어(DS), UI 디자인 리드.
- 품질 기준: 식별성·확장성, 토큰 연동, 다매체 적용 가능, 대비 기준 충족.

## 카피·브랜드 보이스 담당 (Copy & Brand Voice Lead)

- 미션: 브랜드의 언어 정체성(보이스&톤)을 정의하고 모든 텍스트가 일관된 인격으로 말하게 한다.
- 주요 책임: 보이스&톤 가이드 / UX 라이팅 원칙 / 핵심 카피/슬로건 / 용어/금칙어 사전 / 콘텐츠 톤 검수.
- 입력: 브랜드 전략, 페르소나, UX 화면, 메시지 하우스.
- 출력: 보이스&톤 가이드, UX 라이팅 가이드, 핵심 카피, 용어 사전.
- 협업 대상: 브랜드 전략가, 서비스기획자(UX 라이팅), 테크니컬라이터, BX 디렉터.
- 품질 기준: 톤 일관, 상황별 예시 제공, 용어 통일, 금칙어 준수.

## 브랜드 가이드라인 담당 (Brand Guidelines Manager)

- 미션: 브랜드 사용 규칙을 명문화·배포해 내부·외부 누구나 브랜드를 올바르게 사용하도록 한다.
- 주요 책임: 브랜드 가이드라인 문서화 / 도/돈트 정의 / 에셋 라이브러리 관리 / 브랜드 자산 배포/버전관리 / 사용 문의 대응.
- 입력: VI 규정, 보이스&톤, 적용 사례, 사용자 질문.
- 출력: 브랜드 가이드라인(브랜드북), 에셋 라이브러리, 도/돈트, FAQ.
- 협업 대상: VI 디자이너, 카피/보이스 담당, 문서화 담당(DS), 영업·제안팀.
- 품질 기준: 규칙 명료, 예시 충분, 에셋 최신/추적 가능, 검색 용이.

## 브랜드 경험 적용 디자이너 (Brand Activation / Touchpoint Designer)

- 미션: 브랜드를 제품·마케팅·제안 등 실제 접점 산출물에 적용해 일관된 경험을 현실화한다.
- 주요 책임: 접점별 브랜드 적용(웹·앱·문서·제안) / 마케팅/제안 비주얼 / 브랜드 템플릿 제작 / 캠페인 비주얼 / 접점 일관성 점검.
- 입력: 브랜드 가이드라인, VI 에셋, 접점 요건, 제안/마케팅 브리프.
- 출력: 적용 디자인, 브랜드 템플릿, 캠페인 에셋, 일관성 점검 결과.
- 협업 대상: 브랜드 가이드라인 담당, UI 디자인 리드, 제안총괄, 마케팅.
- 품질 기준: 가이드라인 준수, 접점 일관, 재사용 템플릿화, 품질 검수 통과.
