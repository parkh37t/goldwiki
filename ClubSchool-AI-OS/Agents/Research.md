# Research — 리서치팀 역할 카탈로그

> 시장·경쟁·사용자·업종에 대한 근거를 생산하는 사람용 역할 카탈로그입니다. 실행 정본은
> `../.claude/agents/industry-research-lead.md`, 거버넌스·표준 정본은
> [GoldWiki/Research](../GoldWiki/Research/README.md) 및 [GoldWiki/Industry](../GoldWiki/Industry/README.md)입니다.

## 팀 개요

리서치팀은 조직의 모든 의사결정에 검증 가능한 근거를 공급합니다. 시장 규모·트렌드, 경쟁사
전략, 업종 특수성, 실제 사용자 니즈를 조사해 인사이트로 압축하며, 모든 결과는 출처와 함께
[Research](../GoldWiki/Research/README.md)·[ProjectMemory](../GoldWiki/ProjectMemory/README.md)에 축적해
재사용합니다. 추측이 아닌 데이터로 말하는 것이 팀의 원칙입니다.

- 핵심 산출물: 시장 분석, 경쟁 분석, 업종 인사이트, 사용자 리서치 리포트, 근거 라이브러리
- 주요 협업 팀: 전략기획, 영업·제안팀, 비즈니스 컨설팅팀, 프로덕트·UX팀
- 핵심 문서: [Research](../GoldWiki/Research/README.md), [Industry](../GoldWiki/Industry/README.md), [Business](../GoldWiki/Business/README.md)

---

## 시장 리서처 (Market Researcher)

- 미션: 시장 규모·성장성·트렌드를 정량/정성으로 규명해 사업·제안 의사결정의 근거를 만든다.
- 주요 책임: 시장 규모(TAM/SAM/SOM) 추정 / 트렌드·드라이버 분석 / 세그먼트 정의 / 수요 신호 수집 / 출처 신뢰도 평가.
- 입력: 산업 보고서, 공개 통계, 검색·소셜 데이터, 고객 가설.
- 출력: 시장 분석 리포트, 시장 규모 모델, 트렌드 맵, 세그먼트 정의.
- 협업 대상: 전략기획 책임자, 경쟁분석가, BA, 가치설계자.
- 품질 기준: 출처 명시·검증, 추정 방법 투명, 가정·한계 기록.

## 업종 전문가 (Industry Specialist)

- 미션: 특정 업종의 구조·규제·관행·핵심 KPI를 깊이 분석해 업종 맞춤 전략의 토대를 제공한다.
- 주요 책임: 업종 밸류체인·규제 분석 / 업종 표준 KPI·벤치마크 / 업종 RFP 패턴 정리 / 업종별 리스크 식별 / [Industry 문서](../GoldWiki/Industry/README.md) 기여.
- 입력: 업종 규제·보고서, 발주 이력, 현업 인터뷰, 경쟁 사례.
- 출력: 업종 인사이트 리포트, 업종 KPI 벤치마크, 업종 RFP 패턴, 리스크 맵.
- 협업 대상: 도메인전문가, RFP 분석 리드, 시장 리서처, 제안총괄.
- 품질 기준: 규제 최신성, 벤치마크 출처 명확, 업종 특수성 반영.

## 경쟁 분석가 (Competitive Analyst)

- 미션: 경쟁사의 제품·전략·가격·강약점을 분석해 차별화와 win theme의 근거를 제공한다.
- 주요 책임: 경쟁 구도 매핑 / 제품/기능 벤치마킹 / 가격·포지셔닝 분석 / SWOT·격차 분석 / 차별화 기회 도출.
- 입력: 경쟁사 공개 자료, 제품 체험, 가격 정보, 시장 분석.
- 출력: 경쟁 분석 리포트, 기능 비교 매트릭스, 포지셔닝 맵, 차별화 권고.
- 협업 대상: 시장 리서처, 제안총괄, 가격전략가, 프로덕트 오너.
- 품질 기준: 비교 기준 일관, 출처 명확, 차별화 권고는 실행 가능.

## 사용자 리서처 (User Researcher)

- 미션: 실제 사용자의 니즈·행동·페인포인트를 조사해 제품·UX 의사결정에 사람의 진실을 공급한다.
- 주요 책임: 인터뷰/설문/관찰 설계·수행 / 페르소나·여정 입력 / 페인포인트·니즈 분석 / 인사이트 도출 / 가설 검증.
- 입력: 리서치 질문, 사용자 풀, 기존 사용 데이터, 가설.
- 출력: 사용자 리서치 리포트, 페인포인트 맵, 인사이트, 페르소나 입력자료.
- 협업 대상: UX 리서처(프로덕트·UX팀), 서비스기획자, BA, 가치설계자.
- 품질 기준: 표본·방법 명시, 편향 통제, 인사이트는 증거 인용.

## 데스크 리서처 (Desk Researcher / Secondary Research)

- 미션: 2차 자료를 빠르고 정확하게 수집·정제해 다른 리서치의 토대와 근거 라이브러리를 구축한다.
- 주요 책임: 자료 탐색·수집 / 출처 신뢰도 평가 / 핵심 정보 요약·정리 / 근거 라이브러리 구축 / 인용·출처 관리.
- 입력: 리서치 주제, 검색 범위, 신뢰도 기준.
- 출력: 자료 요약집, 근거 라이브러리, 인용 목록, 사실 검증 메모.
- 협업 대상: 시장 리서처, 업종 전문가, 경쟁 분석가, 문서/골드위키 리드.
- 품질 기준: 출처 추적 가능, 1차/2차 구분, 사실/의견 분리, 최신성 확인.

## 리서치 신디사이저 (Research Synthesizer / Insight Lead)

- 미션: 흩어진 조사 결과를 통합해 의사결정자가 즉시 행동할 수 있는 핵심 인사이트로 압축한다.
- 주요 책임: 다출처 데이터 통합 / 패턴·시사점 도출 / 인사이트 우선순위화 / 의사결정 권고 작성 / 리서치 품질 검수.
- 입력: 시장/경쟁/사용자/업종 리서치, 가설, 의사결정 질문.
- 출력: 통합 인사이트 리포트, 의사결정 권고, 핵심 발견 요약, 후속 리서치 제안.
- 협업 대상: 전략기획 책임자, 제안총괄, 프로덕트 오너, BA.
- 품질 기준: 인사이트는 근거 연결, 권고 실행 가능, 상충 데이터 명시.
