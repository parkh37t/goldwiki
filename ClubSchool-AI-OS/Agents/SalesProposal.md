# SalesProposal — 영업·제안팀 역할 카탈로그

> RFP 분석부터 제안서 제출까지를 책임지는 사람용 역할 카탈로그입니다. 실행 정본은
> `../.claude/agents/rfp-strategy-lead.md`, `../.claude/agents/proposal-lead.md`,
> 거버넌스·표준 정본은 [GoldWiki/RFP](../GoldWiki/RFP/RFPAnalysisFramework.md) 및
> [GoldWiki/Proposal](../GoldWiki/Proposal/README.md)입니다.

## 팀 개요

영업·제안팀은 RFP를 해독해 승리 전략(win theme)을 세우고, 평가 기준에 정렬된 제안서를
기한 내 제출합니다. 수주 가능성·마진·리스크를 동시에 최적화하며, 모든 작업은
[RFP 분석 프레임워크](../GoldWiki/RFP/RFPAnalysisFramework.md)와 제안 표준을 먼저 참조하고
결과는 [ProjectMemory](../GoldWiki/ProjectMemory/README.md)에 축적합니다.

- 핵심 산출물: RFP 분석서, win theme, 제안서, 가격 제안, 컴플라이언스 매트릭스
- 주요 협업 팀: 경영/리더십팀(go/no-go), 비즈니스 컨설팅팀, 리서치팀, CTO 산하 기술 리드
- 핵심 문서: [RFP](../GoldWiki/RFP/RFPAnalysisFramework.md), [Proposal](../GoldWiki/Proposal/README.md), [Industry](../GoldWiki/Industry/README.md)

## 제안총괄 (Proposal Director / Proposal Lead)

- 미션: RFP에 정렬된 승리 전략과 제안서 전체 품질·일관성을 책임지고 적기 제출을 보장한다.
- 주요 책임: win theme·차별화 메시지 정의 / 제안 스토리라인 설계 / 섹션 통합·일관성 검수 / go/no-go 안건 작성 / 제출 최종 점검.
- 입력: RFP 원문, 평가 기준, 고객/업종 인사이트, 리서치·기술·가격 입력.
- 출력: 제안 전략서, 제안서 마스터, win theme, 제출본, go/no-go 권고.
- 협업 대상: 사업총괄, 제안PM, 테크니컬라이터, 가격전략가, 컴플라이언스 담당.
- 품질 기준: 평가 기준 100% 대응, 메시지 일관, 근거·차별점 명확, 기한 준수.

## RFP 분석·전략 리드 (RFP Strategy Lead)

- 미션: RFP를 구조적으로 해독해 요구사항·평가 배점·숨은 리스크를 빠짐없이 추출하고 수주 전략의 토대를 만든다.
- 주요 책임: 요구사항 분해·매핑 / 평가 배점 분석 / 경쟁 구도·고객 핫버튼 파악 / 리스크/제약 식별 / 전략 가설 수립.
- 입력: RFP·과업지시서, 사전 질의응답, 발주처 이력, [업종 문서](../GoldWiki/Industry/README.md).
- 출력: RFP 분석서, 요구사항 추적표, 평가 배점 매트릭스, 리스크 리스트.
- 협업 대상: 제안총괄, 비즈니스 분석 리드, 가격전략가, 컴플라이언스 담당.
- 품질 기준: 누락 요구사항 0건, 배점 가중 반영, 가설은 근거 기반.

## 제안 PM (Proposal Project Manager)

- 미션: 제안 작성 전 과정을 일정·인력·버전으로 통제해 마감 사고 없이 고품질 제출을 운영한다.
- 주요 책임: 제안 일정/RACI 운영 / 섹션별 담당·마감 관리 / 리뷰(핑크/레드팀) 운영 / 버전·산출물 형상관리 / 제출 물류.
- 입력: 제안 전략서, 섹션 분담, 마감 일정, 리뷰 피드백.
- 출력: 제안 작업계획, 진척 리포트, 리뷰 일정, 형상관리된 제출본.
- 협업 대상: 제안총괄, PMO 디렉터, 테크니컬라이터, 컴플라이언스 담당.
- 품질 기준: 마감 준수, 단일 정본 유지, 리뷰 게이트 통과 후 제출.

## 테크니컬 라이터 (Technical Writer / Proposal Writer)

- 미션: 복잡한 기술·솔루션을 평가자가 즉시 이해하고 가점할 수 있는 명료한 제안 문장으로 변환한다.
- 주요 책임: 기술 섹션 집필 / 솔루션 내러티브·다이어그램 / 용어·문체 일관성 / 증빙(레퍼런스·실적) 통합 / 가독성 최적화.
- 입력: 기술 리드 자료, 아키텍처 안, win theme, 스타일 가이드.
- 출력: 기술 제안 원고, 솔루션 다이어그램, 실적 요약, 요약문(executive summary).
- 협업 대상: CTO/기술 리드, 제안총괄, BX 카피라이터, 컴플라이언스 담당.
- 품질 기준: 평가 기준 매핑, 주장-근거 일관, 표/다이어그램 가독성, 오탈자 0.

## 가격전략가 (Pricing Strategist)

- 미션: 수주 가능성과 마진을 동시에 최적화하는 가격·견적 전략을 설계한다.
- 주요 책임: 원가 산정·마진 시뮬레이션 / 경쟁 가격 포지셔닝 / 옵션/단계별 견적 설계 / 가격 리스크(범위·변동) 관리 / 협상 카드 준비.
- 입력: 공수 산정, 경쟁 정보, 마진 기준선, 발주 예산 추정.
- 출력: 가격 제안서, 원가/마진 모델, 시나리오별 견적, 협상 가이드.
- 협업 대상: 사업총괄, 제안총괄, PMO(공수), 비즈니스 분석 리드.
- 품질 기준: 마진 기준선 준수, 가정 명시, 시나리오·민감도 포함.

## 컴플라이언스 담당 (Proposal Compliance Specialist)

- 미션: 제안서가 발주 형식·자격·법규 요건을 100% 충족하도록 보증해 형식 탈락을 원천 차단한다.
- 주요 책임: 컴플라이언스 매트릭스 작성·검증 / 제출 형식·서식 점검 / 자격·인증·증빙 확인 / 계약 조건/리스크 조항 리뷰 / 최종 적격성 사인오프.
- 입력: RFP 형식 요건, 자격 기준, 법규/계약 조항, 제출 양식.
- 출력: 컴플라이언스 매트릭스, 적격성 체크리스트, 형식 검수 결과, 리스크 조항 메모.
- 협업 대상: 제안PM, 거버넌스 책임자, 제안총괄, 가격전략가.
- 품질 기준: 형식 요건 0 누락, 증빙 일치, 제출 전 사인오프 완료.
