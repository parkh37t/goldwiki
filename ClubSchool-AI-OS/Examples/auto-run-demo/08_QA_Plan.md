## 0. 경영진 요약 (1페이지)

본 QA 계획은 전국 청소년 동아리 통합 플랫폼(회원 30만·동아리 1.2만, 12억/9개월, 정성 95점 우위형)의 품질을 **요구 단계부터 설계**하기 위한 단일 기준이다. 테스트는 검사가 아니라 위험 통제 활동이며, 본 사업의 4대 품질 위험 — **(1) 아동 개인정보·PIA 컴플라이언스(RISK-01), (2) 지원금 마감일 동시접속 피크(RISK-02), (3) 30만 회원 무손실 이관(RISK-03), (4) 9개월 고정 오픈일(RISK-04)** — 에 검증 자원을 집중 배치한다.

| 핵심 지표 | 목표 | 근거 |
| --- | --- | --- |
| 요구 추적성(RTM) | REQ→SCR→FLOW→TC **100% 무결손** | QualityReviewChecklist §2, 22개 요구 전수 |
| 테스트 케이스 실행률 / 통과율 | ≥ 95% / ≥ 98% | [30 §11] 종료기준 |
| S1·S2 결함 잔존 | **0건** | [30 §10·§11] |
| 핵심 로직 커버리지(인증·정산·서명) | 라인 ≥ 90% / 분기 ≥ 85% | [30 §4] |
| 접근성 | KWCAG 2.2(=WCAG 2.2) AA, axe **위반 0**(critical/serious) | REQ-013, [16] |
| 성능(마감 피크) | 신청 제출 p95 < 2초·실패율 < 0.5%·가용성 99.9% | REQ-019, [07 §5.1] |
| 데이터 이관 | 정합률 **100%** 검증 통과 전 컷오버 금지(게이트) | REQ-016, [07 §4] |

**의사결정 요청:** (a) 마감일 부하 테스트의 **조기 수행(단계 17 이전)** 승인, (b) 데이터 이관 **정합률 100% 컷오버 게이트** 채택, (c) 아동 동의 플로우 **외부 보안 감사** 예산 승인. 본 3건은 높음 등급 리스크 직결 사항으로 [의사결정 로그](../../GoldWiki/32_DECISION_LOG.md) 기록 대상이다.

### 6.1 CI 게이트 ([30 §7])

## 8. 종료기준 (Exit Criteria · 릴리스 게이트)

릴리스(정식 오픈) 전 다음을 **모두** 충족해야 한다([30 §11], [29], QualityReviewChecklist 10단계).

- [ ] 계획된 테스트 케이스 실행률 **≥ 95%**, 통과율 **≥ 98%**
- [ ] **S1·S2 결함 0건**(잔존 없음), S3는 합의된 처리 계획 보유
- [ ] 커버리지 목표(§6.2) 충족 — 핵심 로직 라인 ≥ 90%/분기 ≥ 85%, 전체 ≥ 80%/75%
- [ ] 접근성: axe critical/serious 위반 0, KWCAG 2.2 AA 수동 점검 통과(TC-113)
- [ ] 성능: 마감 피크 p95 < 2초·실패율 < 0.5%·가용성 99.9%, CWV 목표 통과(TC-119/114)
- [ ] 보안: OWASP·DAST·시크릿 스캔 통과, PIA 5종 완비, 아동 동의 감사 통과(§5.2)
- [ ] 데이터 이관: 정합률 **100%** 검증 통과(미만 시 컷오버 차단), 롤백 런북 검증(TC-116)
- [ ] RTM 무결손(REQ→SCR→FLOW→TC 100%), 의무 요구 누락 0
- [ ] [품질 체크리스트(29)] DoD 및 QualityReviewChecklist 10단계 전 단계 Pass
- [ ] 회귀 스위트(P0 24화면) 통과, 플레이키 테스트 0(또는 격리·기록)

> **게이트 판정:** 위 전 항목 Pass 시 통과. 경미 결함(S4)만 잔존 시 Conditional(보완 후 재검). S1/S2 또는 정합률<100% 또는 의무 요구 누락 시 즉시 반려(Fail).

본 QA 계획은 단계 18 [QA 게이트]와 단계 19~21 [안정화·납품]의 실행 기준으로 인계된다. 게이트 판정 결과·반복 결함은 [DecisionLog](../../GoldWiki/DecisionLog/)와 [39 공통 오류](../../GoldWiki/39_COMMON_ERRORS.md)에 누적해 조직 학습으로 환원한다.

**거버넌스 갱신 트리거(의사결정 로그·프로젝트 메모리 기록 대상):**
- 마감일 부하 테스트 **조기 수행**(단계 17 이전) 채택 — RISK-02
- 데이터 이관 **정합률 100% 컷오버 게이트** 채택 — RISK-03
- 아동 동의 플로우 **외부 보안 감사** 편성 — RISK-01
- 종료기준(실행률 95%·통과율 98%·S1/S2 0·커버리지·접근성·성능·보안·이관) 확정

> **거버넌스:** 골드위키 규칙에 따라, 본 문서의 모든 의사결정은 [의사결정 로그](../../GoldWiki/32_DECISION_LOG.md), [프로젝트 메모리](../../GoldWiki/35_PROJECT_MEMORY.md), [베스트 프랙티스](../../GoldWiki/37_BEST_PRACTICES.md), [레퍼런스 라이브러리](../../GoldWiki/36_REFERENCE_LIBRARY.md)를 갱신한다.
