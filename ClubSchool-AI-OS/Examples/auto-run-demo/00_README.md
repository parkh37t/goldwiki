## (b) 한 장 경영 요약

**사업의 본질.** 17개 시·도에 분산된 청소년 동아리의 모집·활동기록·생활기록부 연계 증빙·지원금 신청/심사/정산을 **청소년이 모바일로 체감하는 하나의 통합 포털**로 잇고, 이를 **신학기 고정 오픈일** 안에 안정적으로 여는 것이다. 발주처가 반복 강조하는 무게중심은 (a) 모바일 체감 사용성, (b) KWCAG 2.2 AA로 표현된 디지털 포용, (c) 운영기관 대시보드로 표현된 행정 효율·데이터 가시성의 세 축이다.

**우리의 승부수 — 윈테마 3종(02).**
1. **WT-01 모바일 체감 사용성 × 디지털 포용** — 가입 3클릭 이내, KWCAG 2.2 AA 전 화면 100%(토큰 단계 접근성 내장 + axe 자동점검·수동점검 병행, TC-113·114).
2. **WT-02 9개월 고정 오픈일 무위험 완수 × 무손실 전환** — 오픈일 역산 WBS·범위 동결 게이트, 회원 30만·동아리 1.2만 정합률 100% 검증·롤백·병행운영(TC-116·120).
3. **WT-03 마감일 피크 무장애 × 아동 개인정보 무결성** — 캐싱·CDN·대기열·오토스케일 4중 방어와 마감일 가정 부하 테스트, 만 14세 미만 법정대리인 동의 별도 화면·암호화·PIA(TC-108·119).

**선제 리스크 통제.** 발주처가 가장 우려할 3대 위험을 먼저 통제했다 — ① 고정 오픈일(범위 동결 게이트·MVP 우선순위), ② 30만 회원 이관 손실(정합률 100%·롤백·병행운영), ③ 마감일 동시접속 장애(대기열·오토스케일·조기 부하 테스트). 외부 의존(행정망·본인확인)은 1차월 규격 확정·어댑터·mock 우선 개발로 차단한다.

**경영진 결론.** 본 산출물군은 **가격이 아니라 정성(기술 30·사업이해 20·UX 20 = 변별 핵심 70점)에서 승부**하도록 정렬되어 있다. 평가위원 채점 87.65점(우수 상단)·품질 게이트 94.0%(Conditional Pass, 치명 결함 0)로, **저비용·고효과 보완 소수만으로 탁월(90+)·안전권 진입**이 가능한 상태다.

### 즉시 조치 (제출 전 필수 — D-day 게이트)

1. **P0 채점 견인 2건 반영(최고 ROI).** FIX-01(피크 동시접속 절대 목표치·p95/실패율/가용성·부하 도구·수행 시점 1개 표) → 07 §5.1, FIX-02(PM·CTO·DB Architect·접근성 리드 4인 전담률 100%·투입 타임라인·동종 경력) → 04 §7. 두 건만으로 가중 총점 +1.95 → **89.6점(우수 정점)**, 수주 확률 +6~10%p.
2. **P1 추적성 정합 2건 정정(게이트 통과 조건).** D-01·D-02 — 07·08의 stale 화면 ID를 **05 화면목록 정본** 기준으로 전면 재대조. 정본은 05로 고정.
3. **P1 임원 의사결정 요약 추가.** D-03 — 08 §0 의사결정 요청 3건(조기 부하테스트·이관 100% 게이트·아동동의 외부감사)을 03 경영 요약 또는 별도 0.5p 의사결정 표로 승격.

### 제출 전 권장 (탁월·완결성 진입)

4. **증빙 등급 상향(P2).** FIX-03(접근성 axe 리포트 샘플·가입 3클릭 프로토타입 클릭수, 06)·FIX-05(동종 사업 실측 KPI 인용으로 추정치 대체, 01/03)로 "방법론" → "실증"으로 증거 등급 상향.
5. **운영·가격 정량화(P2).** FIX-04 운영 SLA 표(가용성 99.9%·RTO≤1h·응답 p95·하자보수 KPI)·FIX-06 공수→금액 환산표(420MD 단가 환산, 저배점이나 가격위원 합리성 확인 근거)·D-05 가격 산정 요약표.
6. **외부 의존 공식화(P2).** FIX-07/D-04 — 행정망 연계·병행운영 기간·전자서명 수단·제출 형식 미확정 4건을 발주처 사전 질의서로 공식화하고 미확정 시 우리측 흡수 한도 표로 약속(RISK-03·05 연계).

### 게이트·승인 흐름
