## 1페이지 요약 (단계 3)

전국 17개 시·도에 흩어진 청소년 동아리 정보·모집·활동기록·지원금 신청이 **기관별 자체 시스템·엑셀·종이 서식**으로 분산되어, 청소년 참여 진입장벽이 높고 운영기관의 행정 부담이 크다. 본 사업의 본질은 **모집 → 활동기록 → 생활기록부 연계 증빙 → 지원금 신청·심사·정산**의 전체 흐름을 하나의 모바일 우선 포털로 통합하는 것이다.

RFP가 반복·강조하는 무게중심은 세 축이다. (a) **청소년 70% 이상 모바일 접속**을 전제한 사용성, (b) **KWCAG 2.2 AA·반응형**으로 표현된 디지털 포용·접근성, (c) **지원금 전자화와 운영기관 대시보드**로 표현된 행정 효율·데이터 가시성이다.

**우리 승부수:** 검증된 디자인 시스템·접근성 자산 재사용으로 9개월 고정 일정의 위험을 낮추고, 지원금 마감일 동시접속 피크에 대한 성능 설계와 30만 회원 무손실 이관 방법론을 정량 근거로 제시하여 정성 평가에서 차별화한다.
