# UI 컨셉 템플릿 (UI Concept Template)

> **용도**: 서비스의 비주얼 방향성·무드·레이아웃·핵심 화면 컨셉을 정의해 디자인 시스템과 실제 디자인의 출발점으로 삼는다.
> **사용 에이전트**: ui-design-lead(주), bx-design-lead, ux-research-lead.
> **선행 산출물**: [`UX_Strategy_Template.md`](UX_Strategy_Template.md) · [`ScreenList_Template.md`](ScreenList_Template.md)
> **후속 산출물**: [`DesignSystem_Template.md`](DesignSystem_Template.md)
> **관련 GoldWiki**: [08 UI 가이드라인](../GoldWiki/08_UI_GUIDELINES.md) · [10 Figma 가이드](../GoldWiki/10_FIGMA_GUIDE.md)

### 사용 안내
- 컨셉은 UX 전략·브랜드와 일관돼야 한다(근거를 적는다).
- 키워드 → 비주얼 결정의 **연결 고리**를 명시한다.
- 대표 화면(SCR-ID) 컨셉을 최소 2~3개 기술한다.
