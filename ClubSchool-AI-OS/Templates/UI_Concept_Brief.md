# UI 컨셉 브리프 템플릿 (UI Concept Brief)

> **용도**: 제품의 비주얼 방향성(무드·톤·컬러·타이포·레이아웃·모션)을 정의하여 디자인 시스템 구축과 화면 디자인의 출발점을 만드는 문서. UX 전략을 시각 언어로 번역한다.
> **사용 에이전트**: UI Designer(주), BX Designer, Interaction Designer.
> **선행 산출물**: [`UX_Strategy.md`](UX_Strategy.md).
> **관련 GoldWiki**: [08 UI 가이드라인](../GoldWiki/08_UI_GUIDELINES.md) · [09 디자인 시스템](../GoldWiki/09_DESIGN_SYSTEM.md) · [15 디자인 토큰](../GoldWiki/15_DESIGN_TOKEN.md)

### 작성 팁
- **전략에서 출발한다**: 모든 비주얼 결정은 UX 원칙·브랜드 가치에 근거를 단다.
- **형용사를 시각으로 번역한다**: "신뢰감 있는"을 컬러·여백·타이포의 구체 선택으로 연결한다.
- **레퍼런스는 의도와 함께**: 참고 이미지/사이트는 "무엇을 차용하는가"를 명시한다.
- **토큰을 염두에 둔다**: 컬러/타이포/간격은 디자인 토큰으로 확장 가능하게 정의한다.
