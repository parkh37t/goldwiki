# 최종 결과 보고서 템플릿 (Final Report Template)

> **용도**: 프로젝트 종료 시 목표 대비 성과·산출물·품질·리스크·교훈을 종합 정리해 발주처 제출 및 내부 자산화에 쓴다.
> **사용 에이전트**: executive-director(주), pmo-director, documentation-lead, qa-lead.
> **선행 산출물**: 전 단계 산출물 + [`QA_Checklist_Template.md`](QA_Checklist_Template.md) + [`Risk_Register_Template.md`](Risk_Register_Template.md)
> **관련 GoldWiki**: [31 릴리스 프로세스](../GoldWiki/31_RELEASE_PROCESS.md) · [35 프로젝트 메모리](../GoldWiki/35_PROJECT_MEMORY.md) · [37 베스트 프랙티스](../GoldWiki/37_BEST_PRACTICES.md)

### 사용 안내
- 목표(제안 시 약속) 대비 **실제 성과**를 정량으로 대조한다.
- 요구ID 충족 현황으로 **추적성을 닫아** 완전성을 증명한다.
- 교훈은 프로젝트 메모리·베스트 프랙티스로 환류한다.
