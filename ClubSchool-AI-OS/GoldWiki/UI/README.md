# UI 토픽 — GoldWiki

화면 단위 시각 설계의 단일 진실 공급원(SSOT). UX 전략을 화면으로 구현하기 위한 그리드·타이포·색·상태·반응형 규칙을 모은다.

## 폴더 목적

- 일관·접근 가능·구현 가능한 화면 UI 규칙 제공
- 디자인 토큰을 화면에 적용하는 사용 규칙 규정
- 와이어프레임→UI→핸드오프의 시각 표준화
