# UX 토픽 — GoldWiki

사용자 경험(UX) 전략·리서치·정보구조·플로우의 단일 진실 공급원(SSOT). 비즈니스 목표를 사용자 가치로 번역하고, 근거 기반으로 경험을 설계·측정하는 표준을 모은다.

## 폴더 목적

- 리서치 → 인사이트 → 전략 → 측정의 폐루프 운영
- 정보 구조(IA)와 유저 플로우의 설계 표준 제공
- UI·디자인 시스템·QA로 이어지는 상류 산출물 생산
