# Research — 업종/시장 리서치 토픽

## 폴더 목적

RFP 분석·제안 전략·비즈니스 분석을 뒷받침하는 **업종 특성·시장 동향·사용자·경쟁 리서치**를 보관한다. 발주처의 숨은 기대를 해석하고, 윈 테마와 차별화의 사실 기반을 제공하는 토픽이다. 업종별 심층 자료는 [Industry](../Industry/) 토픽에 정리되어 있으며, 본 토픽은 리서치 방법·시장 분석·진입점을 관리한다.
